# TodoApp Product Spec

## Overview
Describe the TodoApp goal and main users.
## Target Users
* Users who need to manage their tasks
## Main Features
* User management
* Group management
* Role management
* Todo management
## Business Rules
* Users can create, read, update and delete their todos
* Users can be assigned to groups
* Users can have roles
## User Stories
* As a user, I want to create a new todo
* As a user, I want to view all my todos
* As a user, I want to update a todo
* As a user, I want to delete a todo
## Acceptance Criteria
* The app allows users to create, read, update and delete todos
* The app allows users to be assigned to groups
* The app allows users to have roles
## Out of Scope
* Integration with other apps
