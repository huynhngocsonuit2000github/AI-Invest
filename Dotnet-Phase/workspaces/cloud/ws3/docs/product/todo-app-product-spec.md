# Todo App Product Spec

## Overview
Describe the todo app goal and main users.
## Target Users
* Users who want to manage their tasks
## Main Features
* User management
* Group management
* Role management
* Todo management
## Business Rules
* Users can create, read, update and delete todos
* Users can be assigned to groups
* Users can have roles
## User Stories
* As a user, I want to create a new todo
* As a user, I want to view all my todos
* As a user, I want to update a todo
* As a user, I want to delete a todo
## Acceptance Criteria
* Todos are stored in the database
* Users can login and logout
* Users can view their todos
## Out of Scope
* Integration with other apps
