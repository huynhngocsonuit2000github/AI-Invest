# Todo App Architecture

## Overview
The architecture of the Todo App is designed to be scalable, maintainable, and secure. It uses .NET 8 Web API, EF Core for ORM, PostgreSQL as the database, JWT for authentication, and a controller-service-repository structure.

## Components
- **Controllers**: Handle HTTP requests and responses.
- **Services**: Business logic layer.
- **Repositories**: Data access layer.
- **Models**: Entity models and DTOs.
- **Database**: PostgreSQL with EF Core ORM.
