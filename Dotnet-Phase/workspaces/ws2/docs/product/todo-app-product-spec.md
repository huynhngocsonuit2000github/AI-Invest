# Todo App Product Specification

## Overview
The Todo App is a web application that allows users to manage their tasks. It includes features for user management, group management, role-based access control (RBAC), and task management.

## Features
- User Management: Register, login, update profile, change password.
- Group Management: Create, update, delete groups.
- Role Management: Define roles with permissions.
- RBAC: Assign roles to users and groups.
- Task Management: Create, read, update, delete tasks.
