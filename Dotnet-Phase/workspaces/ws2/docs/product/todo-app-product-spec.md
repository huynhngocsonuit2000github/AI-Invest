# TodoApp Product Specification

## Project Overview
This document outlines the product specification for the TodoApp project, which is a .NET 8 Web API application using EF Core and PostgreSQL.

## Requirements
- Use controller-based ASP.NET Core Web API
- Add EF Core packages for PostgreSQL
- Add basic folder structure: Controllers, Services, Repositories, Models, DTOs, Data
- Add AppDbContext
- Add initial Program.cs configuration
- Do not implement user, group, role, or todo CRUD yet
