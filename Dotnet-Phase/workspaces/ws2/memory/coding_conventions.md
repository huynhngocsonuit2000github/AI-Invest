# Coding Conventions

## Naming Conventions
- Entity classes: `User`, `Group`, `Role`, `Task`
- DTO classes: `UserDto`, `GroupDto`, `RoleDto`, `TaskDto`
- Service interfaces and implementations: `IUserService`, `UserService`
- Repository interfaces and implementations: `IUserRepository`, `UserRepository`

## File Structure
- `docs/`: Contains project documentation.
- `scrum/backlog/`: Contains the product backlog for the project.
- `memory/`: Contains knowledge and guidelines for the project.
